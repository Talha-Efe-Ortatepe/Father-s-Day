# Father's Day Heart Animation (Tkinter)

This project shows a heart animation and a loving message for Father's Day using Python's Tkinter library.

## Features

- Animated heart shape
- Shows a random loving message when you click the button
- Simple and easy to use

## How to Run

1. Make sure you have Python 3.x installed.
2. Run the script:
   ```
   python baba.py
   ```

## Requirements

- Python 3.x
- Tkinter (comes with standard Python)
