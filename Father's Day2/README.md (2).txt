# Father's Day Heart Drawing (Turtle)

This project draws a red heart and displays a "Happy Father's Day!" message using Python's Turtle graphics.

## Features

- Draws a heart shape with Turtle
- Shows a special message for Father's Day
- Fun and visual

## How to Run

1. Make sure you have Python 3.x installed.
2. Run the script:
   ```
   python baba2.py
   ```

## Requirements

- Python 3.x
- Turtle (comes with standard Python)
